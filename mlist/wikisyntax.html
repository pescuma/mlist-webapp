Wiki Syntax
(aceito nos campos de texto)

Para começar um novo parágrafo:
- deixe uma linha em branco

Para formatar uma palavra:
(o marcador precisa estar colado na palavra)
/texto/texto
_texto_texto
*texto*texto
~texto~texto
^texto^texto
,,texto,,texto
!!texto!!texto
++texto++texto
--texto--texto

Para inserir uma imagem:
- digite a URL dela

Para inserir um link:
- digite a URL
- [[http://a.com]]
- [[http://a.com][texto do link]]

Para inserir código fonte:
- digite: [code]if (x) { return }[/code]
- se souber a linguagem, digite ela dendo de []: [c++]if (x) { return }[/c++]
